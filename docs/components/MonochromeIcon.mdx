# Monochrome Icon 규칙

컬러 금지, Stroke 2px, 24x24 기준의 모노크롬 아이콘 가이드라인에 대해 설명합니다.

## 개요

하모니 스마트팜 솔루션에서는 일관된 디자인을 위해 모노크롬 아이콘을 사용합니다. 모든 아이콘은 컬러를 사용하지 않고, 2px stroke, 24x24 크기를 기준으로 합니다.

## 아이콘 규칙

### 1. 컬러 금지

#### 규칙
- **단색 사용**: 아이콘은 단일 색상만 사용
- **컬러 금지**: 여러 색상 사용 금지
- **투명도 사용 가능**: 투명도는 사용 가능

#### 예시
- ✅ 단일 색상 아이콘
- ❌ 여러 색상 아이콘
- ✅ 투명도 적용 아이콘

### 2. Stroke 2px

#### 규칙
- **일관된 stroke**: 모든 아이콘은 2px stroke 사용
- **stroke만 사용**: Fill 대신 stroke 사용
- **일관성 유지**: 모든 아이콘에서 동일한 stroke 두께

#### 예시
```svg
<svg width="24" height="24" viewBox="0 0 24 24" fill="none">
  <path 
    d="M12 2L2 7L12 12L22 7L12 2Z" 
    stroke="currentColor" 
    stroke-width="2" 
    stroke-linecap="round" 
    stroke-linejoin="round"
  />
</svg>
```

### 3. 24x24 기준

#### 규칙
- **표준 크기**: 기본 크기는 24x24px
- **viewBox**: viewBox="0 0 24 24" 사용
- **확장 가능**: 필요 시 크기 조절 가능 (비율 유지)

#### 예시
```svg
<svg 
  width="24" 
  height="24" 
  viewBox="0 0 24 24" 
  fill="none"
  xmlns="http://www.w3.org/2000/svg"
>
  <!-- 아이콘 경로 -->
</svg>
```

## 아이콘 라이브러리

### 권장 라이브러리
- **Heroicons**: Tailwind CSS 팀의 아이콘 라이브러리

- **Phosphor Icons**: 다양한 스타일의 모노크롬 아이콘
