# Monitoring

실시간 데이터 스트림 및 상세 모니터링 화면에 대해 설명합니다.

## 개요

Monitoring 화면은 센서 데이터, 장치 상태, 로봇 상태 등을 실시간으로 상세하게 모니터링할 수 있는 화면입니다.

## 주요 기능

### 1. 실시간 데이터 스트림

#### 데이터 스트리밍
- **실시간 업데이트**: 데이터 실시간 업데이트
- **고주파 업데이트**: 초 단위 업데이트
- **WebSocket**: 양방향 실시간 통신

#### 표시 방식
- **숫자 표시**: 현재 값 숫자로 표시
- **게이지**: 원형 또는 막대 게이지
- **그래프**: 실시간 그래프
- **상태 표시**: 정상/주의/위험 상태 표시

### 2. 센서 모니터링

#### 환경 센서
- **온도**: 실시간 온도 값 및 그래프
- **습도**: 실시간 습도 값 및 그래프
- **CO₂**: 실시간 CO₂ 농도 및 그래프
- **조도**: 실시간 조도 값 및 그래프

#### 작물 센서
- **토양 수분**: 실시간 토양 수분 함량
- **토양 EC**: 실시간 토양 EC 값
- **엽면 습도**: 실시간 엽면 습도
- **줄기 직경**: 실시간 줄기 직경

#### 센서 상세 정보
- **센서 위치**: 센서 설치 위치
- **센서 상태**: 센서 연결 상태
- **마지막 업데이트**: 마지막 데이터 수신 시간
- **센서 설정**: 센서 설정 정보

### 3. 장치 모니터링

#### 환경 제어 장치
- **난방기**: 난방기 ON/OFF 상태, 출력
- **환기팬**: 환기팬 ON/OFF 상태, 속도
- **스크린**: 스크린 개폐 상태, 개폐율
- **CO₂ 공급**: CO₂ 공급 ON/OFF 상태, 유량
- **가습기/제습기**: 가습/제습 상태
- **인공광**: 인공광 ON/OFF 상태, 밝기

#### 관수 시스템
- **관수 밸브**: 관수 밸브 ON/OFF 상태
- **관수 펌프**: 관수 펌프 ON/OFF 상태, 속도
- **관수량**: 현재 관수량

#### 장치 상세 정보
- **장치 위치**: 장치 설치 위치
- **장치 상태**: 장치 작동 상태
- **에너지 소비**: 장치 에너지 소비량
- **작동 시간**: 장치 작동 시간

### 4. 로봇 모니터링

#### 로봇 상태
- **위치**: 로봇 현재 위치
- **작업 상태**: 로봇 작업 상태 (대기/작업 중/충전 중)
- **배터리**: 로봇 배터리 상태
- **속도**: 로봇 이동 속도

#### 작업 정보
- **현재 작업**: 현재 수행 중인 작업
- **작업 진행률**: 작업 진행률
- **예상 완료 시간**: 예상 작업 완료 시간
- **작업 이력**: 최근 작업 이력

### 5. 그래프 및 차트

#### 시계열 그래프
- **단일 센서**: 단일 센서 시계열 그래프
- **다중 센서**: 여러 센서 비교 그래프
- **기간 선택**: 시간 범위 선택 (1시간, 6시간, 24시간, 7일, 30일)
- **줌/팬**: 그래프 확대/축소 및 이동

#### 통계 차트
- **평균값**: 기간별 평균값
- **최대/최소값**: 기간별 최대/최소값
- **분포**: 값 분포 히스토그램
- **상관관계**: 센서 간 상관관계

### 6. 알람 모니터링

#### 실시간 알람
- **알람 목록**: 현재 발생한 알람 목록
- **알람 상세**: 알람 상세 정보
- **알람 처리**: 알람 확인/해결 처리

#### 알람 이력
- **알람 이력**: 과거 알람 이력
- **알람 통계**: 알람 발생 통계
- **알람 분석**: 알람 패턴 분석

## 화면 구성

### 1. 센서 목록
- **센서 그룹**: 센서 그룹별 표시
- **센서 카드**: 각 센서별 카드 표시
- **필터**: 센서 타입별 필터
- **검색**: 센서 검색

### 2. 그래프 영역
- **대형 그래프**: 큰 화면에 그래프 표시
- **다중 그래프**: 여러 그래프 동시 표시
- **전체 화면**: 전체 화면 모드

### 3. 상태 패널
- **장치 상태**: 장치 상태 요약
- **로봇 상태**: 로봇 상태 요약
- **시스템 상태**: 시스템 상태 요약

### 4. 알람 패널
- **알람 목록**: 알람 목록 표시
- **알람 필터**: 알람 타입별 필터
- **알람 정렬**: 알람 정렬 옵션

## 필터 및 검색

### 1. 센서 필터
- **센서 타입**: 환경 센서/작물 센서 필터
- **구역**: 온실 구역별 필터
- **상태**: 정상/주의/위험 상태 필터

### 2. 시간 필터
- **빠른 선택**: 1시간, 6시간, 24시간, 7일, 30일
- **사용자 정의**: 시작일/종료일 선택
- **실시간**: 실시간 모드

### 3. 검색
- **센서 검색**: 센서 이름으로 검색
- **장치 검색**: 장치 이름으로 검색
- **로봇 검색**: 로봇 이름으로 검색

## 데이터 내보내기

### 1. 데이터 다운로드
- **CSV**: CSV 형식으로 다운로드
- **Excel**: Excel 형식으로 다운로드
- **JSON**: JSON 형식으로 다운로드

### 2. 리포트 생성
- **일일 리포트**: 일일 모니터링 리포트
- **주간 리포트**: 주간 모니터링 리포트
- **월간 리포트**: 월간 모니터링 리포트

## 성능 최적화

### 1. 데이터 최적화
- **샘플링**: 대량 데이터 샘플링
- **집계**: 데이터 집계
- **캐싱**: 자주 조회되는 데이터 캐싱

### 2. 렌더링 최적화
- **가상 스크롤**: 대량 데이터 가상 스크롤
- **지연 로딩**: 필요 시 데이터 로딩
- **그래프 최적화**: 그래프 렌더링 최적화
