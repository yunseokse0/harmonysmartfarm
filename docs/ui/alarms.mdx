# 알람 시스템

임계값 기반 자동 알람 및 알람 관리 시스템에 대해 설명합니다.

## 개요

알람 시스템은 센서 데이터, 장치 상태, 시스템 상태를 모니터링하여 이상 상황 발생 시 자동으로 알람을 발생시키고 관리하는 시스템입니다.

## 알람 타입

### 1. 센서 알람

#### 환경 센서 알람
- **온도 알람**: 온도가 임계값을 초과/미만 시 알람
- **습도 알람**: 습도가 임계값을 초과/미만 시 알람
- **CO₂ 알람**: CO₂ 농도가 임계값을 초과/미만 시 알람
- **조도 알람**: 조도가 임계값을 초과/미만 시 알람

#### 작물 센서 알람
- **토양 수분 알람**: 토양 수분이 임계값을 초과/미만 시 알람
- **토양 EC 알람**: 토양 EC가 임계값을 초과/미만 시 알람
- **엽면 습도 알람**: 엽면 습도가 임계값을 초과 시 알람

#### 센서 오류 알람
- **센서 연결 끊김**: 센서 통신 단절 시 알람
- **센서 데이터 이상**: 센서 데이터 이상 시 알람
- **센서 보정 필요**: 센서 보정 필요 시 알람

### 2. 장치 알람

#### 환경 제어 장치 알람
- **장치 오류**: 장치 작동 오류 시 알람
- **장치 연결 끊김**: 장치 통신 단절 시 알람
- **장치 과부하**: 장치 과부하 시 알람

#### 관수 시스템 알람
- **관수 실패**: 관수 실행 실패 시 알람
- **관수량 이상**: 관수량 이상 시 알람
- **펌프 오류**: 관수 펌프 오류 시 알람

### 3. 로봇 알람

#### 로봇 상태 알람
- **로봇 오류**: 로봇 작동 오류 시 알람
- **로봇 연결 끊김**: 로봇 통신 단절 시 알람
- **배터리 부족**: 로봇 배터리 부족 시 알람

#### 작업 알람
- **작업 실패**: 로봇 작업 실패 시 알람
- **작업 지연**: 작업 지연 시 알람
- **충돌**: 로봇 충돌 시 알람

### 4. AI 분석 알람

#### 병충해 알람
- **병충해 탐지**: AI가 병충해 탐지 시 알람
- **병충해 확산**: 병충해 확산 위험 시 알람

#### 작물 상태 알람
- **작물 스트레스**: 작물 스트레스 높을 시 알람
- **생육 이상**: 생육 이상 탐지 시 알람

### 5. 시스템 알람

#### 시스템 오류
- **서버 오류**: 서버 오류 시 알람
- **데이터베이스 오류**: 데이터베이스 오류 시 알람
- **통신 오류**: 통신 오류 시 알람

#### 보안 알람
- **무단 접근**: 무단 접근 시도 시 알람
- **인증 실패**: 인증 실패 시 알람

## 알람 설정

### 1. 임계값 설정

#### 임계값 타입
- **상한 임계값**: 값이 상한을 초과 시 알람
- **하한 임계값**: 값이 하한을 미만 시 알람
- **범위 임계값**: 값이 범위를 벗어날 시 알람

#### 임계값 설정 방법
- **수동 설정**: 사용자가 직접 설정
- **자동 설정**: 시스템이 자동으로 설정
- **템플릿**: 미리 정의된 템플릿 사용

### 2. 알람 심각도

#### 심각도 레벨
- **경미 (Info)**: 정보성 알람, 즉시 조치 불필요
- **주의 (Warning)**: 주의 필요, 모니터링 필요
- **심각 (Critical)**: 즉시 조치 필요

#### 심각도별 동작
- **경미**: 알람 기록만
- **주의**: 알람 표시 및 알림
- **심각**: 알람 표시, 알림, 긴급 통지

### 3. 알람 알림

#### 알림 방식
- **Dashboard 알람**: Dashboard에 알람 표시
- **이메일 알림**: 이메일로 알림 전송
- **SMS 알림**: SMS로 알림 전송
- **푸시 알림**: 모바일 앱 푸시 알림
- **음성 알림**: 음성 알림 (선택)

#### 알림 설정
- **알람 타입별**: 알람 타입별 알림 설정
- **심각도별**: 심각도별 알림 설정
- **시간대별**: 시간대별 알림 설정 (야간 알림 제한)

## 알람 관리

### 1. 알람 목록

#### 알람 표시
- **실시간 알람**: 현재 발생한 알람 목록
- **알람 이력**: 과거 알람 이력
- **필터**: 알람 타입, 심각도, 상태별 필터
- **검색**: 알람 내용 검색

#### 알람 정보
- **알람 타입**: 알람 종류
- **발생 시간**: 알람 발생 시간
- **심각도**: 알람 심각도
- **상태**: 알람 상태 (미확인/확인/해결)
- **상세 정보**: 알람 상세 정보

### 2. 알람 처리

#### 알람 상태
- **미확인**: 알람 발생 후 아직 확인 안 함
- **확인**: 알람 확인함
- **해결**: 알람 원인 해결함
- **무시**: 알람 무시 (경미한 알람)

#### 알람 처리 방법
- **확인**: 알람 확인 처리
- **해결**: 알람 해결 처리
- **주석 추가**: 알람에 주석 추가
- **할당**: 담당자에게 알람 할당

### 3. 알람 통계

#### 통계 정보
- **알람 발생 횟수**: 기간별 알람 발생 횟수
- **알람 타입별 통계**: 알람 타입별 발생 통계
- **심각도별 통계**: 심각도별 발생 통계
- **평균 해결 시간**: 알람 평균 해결 시간

#### 리포트
- **일일 리포트**: 일일 알람 리포트
- **주간 리포트**: 주간 알람 리포트
- **월간 리포트**: 월간 알람 리포트

## 알람 규칙

### 1. 알람 규칙 생성

#### 규칙 구조
```
IF (조건) THEN (알람 발생)
```

#### 예시
- IF 온도 > 35℃ THEN 심각 알람 발생
- IF 토양 수분 < 15% THEN 주의 알람 발생
- IF 병충해 탐지 THEN 심각 알람 발생
