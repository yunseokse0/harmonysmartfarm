# Dashboard

센서 요약, 제어 버튼, 알람 표시를 포함한 통합 대시보드에 대해 설명합니다.

## 개요

Dashboard는 하모니 스마트팜 솔루션의 중앙 제어 화면으로, 실시간 센서 데이터, 제어 인터페이스, 알람 정보를 한눈에 볼 수 있는 통합 인터페이스입니다.

## 주요 구성 요소

### 1. 센서 요약

#### 환경 센서 요약
- **온도**: 현재 온도 및 목표 온도
- **습도**: 현재 습도 및 목표 습도
- **CO₂**: 현재 CO₂ 농도 및 목표 농도
- **조도**: 현재 조도 및 목표 조도

#### 작물 센서 요약
- **토양 수분**: 현재 토양 수분 함량
- **토양 EC**: 현재 토양 EC 값
- **엽면 습도**: 현재 엽면 습도
- **줄기 직경**: 현재 줄기 직경

#### 표시 방식
- **게이지**: 원형 또는 막대 게이지
- **숫자**: 큰 숫자로 현재 값 표시
- **상태 표시**: 정상/주의/위험 상태 표시
- **트렌드**: 최근 변화 트렌드 표시

### 2. 제어 버튼

#### 환경 제어
- **난방기**: 난방기 ON/OFF 버튼
- **환기팬**: 환기팬 ON/OFF 및 속도 조절
- **스크린**: 스크린 개폐 버튼
- **CO₂ 공급**: CO₂ 공급 ON/OFF 버튼
- **가습기/제습기**: 가습/제습 버튼
- **인공광**: 인공광 ON/OFF 및 밝기 조절

#### 관수 시스템
- **관수 밸브**: 관수 밸브 ON/OFF 버튼
- **관수 펌프**: 관수 펌프 ON/OFF 버튼
- **관수량 설정**: 관수량 설정 슬라이더

#### 로봇 제어
- **방제 로봇**: 방제 로봇 작업 실행 버튼
- **수확 로봇**: 수확 로봇 작업 실행 버튼
- **운반 로봇**: 운반 로봇 작업 실행 버튼

### 3. 알람 표시

#### 알람 목록
- **실시간 알람**: 현재 발생한 알람 목록
- **알람 이력**: 과거 알람 이력
- **알람 우선순위**: 알람 우선순위 표시

#### 알람 정보
- **알람 타입**: 알람 종류 (온도, 습도, 병충해 등)
- **알람 시간**: 알람 발생 시간
- **알람 상태**: 알람 상태 (미확인/확인/해결)
- **알람 심각도**: 알람 심각도 (경미/중간/심각)

### 4. 그래프 및 차트

#### 시계열 그래프
- **온도 그래프**: 시간별 온도 변화
- **습도 그래프**: 시간별 습도 변화
- **CO₂ 그래프**: 시간별 CO₂ 농도 변화
- **토양 수분 그래프**: 시간별 토양 수분 변화

#### 비교 차트
- **목표 vs 실제**: 목표 값과 실제 값 비교
- **구역별 비교**: 여러 구역 비교
- **기간별 비교**: 기간별 비교

### 5. 지도 및 레이아웃

#### 온실 레이아웃
- **구역 표시**: 온실 구역 표시
- **센서 위치**: 센서 위치 표시
- **장치 위치**: 제어 장치 위치 표시
- **로봇 위치**: 로봇 현재 위치 표시

#### 상태 표시
- **구역별 상태**: 구역별 환경 상태 표시
- **장치 상태**: 장치 ON/OFF 상태 표시
- **로봇 상태**: 로봇 작업 상태 표시

- **알람 아이콘**: 알람 개수 표시

### 2. 사이드바

  - Monitoring
  - Control
  - Analytics
  - Settings

- **센서 요약 카드**: 주요 센서 값 카드
- **그래프 영역**: 시계열 그래프
- **제어 패널**: 제어 버튼 패널
- **알람 패널**: 알람 목록 패널

### 4. 푸터
- **시스템 상태**: 시스템 상태 정보

- **업데이트 시간**: 마지막 업데이트 시간
