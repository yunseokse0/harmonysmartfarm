# 분석 리포트

일/주/월 단위 통계 및 분석 리포트에 대해 설명합니다.

## 개요

분석 리포트는 센서 데이터, 환경 제어, 작물 생육, 로봇 작업 등을 분석하여 일/주/월 단위로 리포트를 생성하는 시스템입니다.

## 리포트 타입

### 1. 일일 리포트

#### 리포트 내용
- **환경 요약**: 일일 평균/최대/최소 온도, 습도, CO₂, 조도
- **작물 상태**: 일일 작물 상태 요약
- **제어 활동**: 일일 제어 활동 요약
- **에너지 소비**: 일일 에너지 소비량
- **알람 요약**: 일일 알람 발생 요약

#### 리포트 형식
- **요약 카드**: 주요 지표 요약 카드
- **그래프**: 일일 데이터 그래프
- **테이블**: 상세 데이터 테이블
- **비교**: 전일 대비 비교

### 2. 주간 리포트

#### 리포트 내용
- **환경 트렌드**: 주간 환경 변화 트렌드
- **작물 생육**: 주간 작물 생육 변화
- **제어 효율**: 주간 제어 효율 분석
- **에너지 분석**: 주간 에너지 소비 분석
- **알람 분석**: 주간 알람 발생 분석

#### 리포트 형식
- **트렌드 그래프**: 주간 트렌드 그래프
- **통계 차트**: 주간 통계 차트
- **비교 분석**: 전주 대비 비교 분석
- **요약**: 주간 요약 정보

### 3. 월간 리포트

#### 리포트 내용
- **환경 종합**: 월간 환경 종합 분석
- **작물 생육 종합**: 월간 작물 생육 종합
- **제어 종합**: 월간 제어 종합 분석
- **에너지 종합**: 월간 에너지 소비 종합
- **비용 분석**: 월간 운영 비용 분석
- **생산성 분석**: 월간 생산성 분석

#### 리포트 형식
- **종합 대시보드**: 월간 종합 대시보드
- **상세 분석**: 월간 상세 분석
- **비교 분석**: 전월 대비 비교 분석
- **예측**: 다음 달 예측 정보

## 분석 항목

### 1. 환경 분석

#### 온도 분석
- **평균 온도**: 기간별 평균 온도
- **최대/최소 온도**: 기간별 최대/최소 온도
- **온도 분포**: 온도 분포 히스토그램
- **목표 달성률**: 목표 온도 달성률

#### 습도 분석
- **평균 습도**: 기간별 평균 습도
- **최대/최소 습도**: 기간별 최대/최소 습도
- **습도 분포**: 습도 분포 히스토그램
- **목표 달성률**: 목표 습도 달성률

#### CO₂ 분석
- **평균 CO₂**: 기간별 평균 CO₂ 농도
- **CO₂ 공급 시간**: CO₂ 공급 시간
- **CO₂ 효율**: CO₂ 활용 효율

#### 조도 분석
- **평균 조도**: 기간별 평균 조도
- **일사량**: 일일 일사량
- **인공광 사용**: 인공광 사용 시간 및 에너지

### 2. 작물 분석

#### 생육 분석
- **생육 속도**: 생육 속도 분석
- **생육 단계**: 생육 단계 진행 상황
- **생육 지표**: 생육 지표 변화

#### 스트레스 분석
- **스트레스 발생**: 스트레스 발생 빈도 및 강도
- **스트레스 원인**: 스트레스 원인 분석
- **스트레스 대응**: 스트레스 대응 효과

#### 병충해 분석
- **병충해 발생**: 병충해 발생 빈도 및 종류
- **방제 효과**: 방제 효과 분석
- **방제 비용**: 방제 비용 분석

### 3. 제어 분석

#### 제어 활동
- **제어 횟수**: 기간별 제어 실행 횟수
- **제어 시간**: 기간별 제어 실행 시간
- **제어 효율**: 제어 목표 달성률

#### 장치 사용
- **장치 사용 시간**: 각 장치 사용 시간
- **장치 사용 빈도**: 각 장치 사용 빈도
- **장치 효율**: 각 장치 효율

### 4. 에너지 분석

#### 에너지 소비
- **총 에너지 소비**: 기간별 총 에너지 소비
- **장치별 소비**: 각 장치별 에너지 소비
- **에너지 트렌드**: 에너지 소비 트렌드

#### 에너지 효율
- **에너지 효율**: 에너지 효율 분석
- **비용 분석**: 에너지 비용 분석
- **절감 효과**: 에너지 절감 효과

### 5. 생산성 분석

#### 생산량
- **생산량**: 기간별 생산량
- **생산량 트렌드**: 생산량 변화 트렌드
- **품질**: 생산물 품질 분석

#### 효율
- **생산 효율**: 생산 효율 분석
- **비용 효율**: 비용 대비 생산량
- **에너지 효율**: 에너지 대비 생산량

## 리포트 생성

### 1. 자동 생성

#### 생성 스케줄
- **일일 리포트**: 매일 자동 생성 (익일 아침)
- **주간 리포트**: 매주 자동 생성 (월요일 아침)
- **월간 리포트**: 매월 자동 생성 (익월 1일)

#### 생성 방식
- **백그라운드**: 백그라운드에서 자동 생성
- **이메일 전송**: 생성된 리포트 이메일 전송
- **저장**: 리포트 자동 저장

### 2. 수동 생성

#### 생성 방법
- **기간 선택**: 리포트 기간 선택
- **항목 선택**: 리포트 항목 선택
- **생성**: 리포트 생성 버튼 클릭

#### 생성 옵션
- **형식 선택**: PDF, Excel, HTML 형식 선택
- **항목 선택**: 포함할 항목 선택
- **비교 기간**: 비교할 기간 선택
