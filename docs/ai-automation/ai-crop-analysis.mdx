# AI 작물 분석

카메라 이미지 기반 작물 생육 상태 분석 및 스트레스 탐지에 대해 설명합니다.

**참고**: 본 문서는 스마트팜코리아 표준 구성요소인 **영상장비**를 활용한 AI 분석을 기준으로 작성되었습니다.

## 개요

AI 작물 분석은 컴퓨터 비전과 딥러닝 기술을 활용하여 작물의 생육 상태를 자동으로 분석하고 평가하는 시스템입니다. 영상장비(CCTV, 웹카메라)에서 수집한 이미지를 기반으로 분석을 수행합니다.

## 주요 기능

### 1. 생육 상태 분석

#### 생육 단계 판별
- **발아 단계**: 종자 발아 상태 확인
- **생육 단계**: 초기 생육, 중기 생육, 후기 생육 구분
- **개화 단계**: 개화 시기 및 개화율 측정
- **결실 단계**: 과실 형성 및 성숙도 판별

#### 생육 지표 측정
- **엽면적**: 잎 면적 측정 (LAI - Leaf Area Index)
- **식물체 높이**: 작물 높이 측정
- **줄기 직경**: 줄기 두께 측정
- **과실 크기**: 과실 크기 및 개수 측정

### 2. 스트레스 탐지

#### 수분 스트레스
- **엽면 온도**: 적외선 이미지를 통한 엽면 온도 측정
- **엽면 색상 변화**: 수분 부족 시 잎 색상 변화 탐지
- **잎 말림**: 수분 스트레스로 인한 잎 말림 탐지

#### 영양 스트레스
- **엽색 변화**: 영양 부족으로 인한 엽색 변화 탐지
- **생육 저하**: 정상 대비 생육 속도 저하 탐지

#### 환경 스트레스
- **온도 스트레스**: 고온/저온 스트레스 탐지
- **광 스트레스**: 과도한 광량 또는 부족 탐지

### 3. 생육 속도 분석

#### 성장률 측정
- **일일 성장률**: 일일 생육 속도 계산
- **누적 생육량**: 누적 생육량 추적
- **예측 생육**: 머신러닝 기반 생육 예측

#### 비교 분석
- **기준 작물과 비교**: 정상 작물과 비교 분석
- **이전 시기와 비교**: 시계열 비교 분석
- **다른 구역과 비교**: 구역별 비교 분석

## 기술 구조

### 이미지 수집 (영상장비)

#### 영상장비 시스템 (스마트팜코리아 표준)
- **CCTV**: 고해상도 영상 모니터링
- **웹카메라**: 실시간 작물 상태 관찰
- **고해상도 카메라**: 4K 이상 해상도
- **멀티스펙트럼 카메라**: 다양한 파장 대역 촬영
- **열화상 카메라**: 적외선 이미지 촬영
- **자동 촬영**: 주기적 자동 촬영 (1일 1회 ~ 1시간 1회)

#### 촬영 조건
- **일정한 위치**: 고정된 위치에서 촬영
- **일정한 각도**: 동일한 각도로 촬영
- **조명 조건**: 일정한 조명 조건 유지

### 이미지 전처리

#### 전처리 과정
- **노이즈 제거**: 이미지 노이즈 제거
- **밝기 보정**: 조명 변화 보정
- **색상 보정**: 색상 일관성 유지
- **크기 정규화**: 이미지 크기 통일

### AI 모델

#### 딥러닝 모델
- **CNN (Convolutional Neural Network)**: 이미지 특징 추출
- **ResNet**: 깊은 신경망 구조
- **EfficientNet**: 효율적인 네트워크 구조
- **Vision Transformer**: 트랜스포머 기반 모델

#### 모델 학습
- **데이터셋**: 수천~수만 장의 작물 이미지
- **라벨링**: 전문가가 생육 상태 라벨링
- **증강**: 데이터 증강 (회전, 크기 조절 등)
- **학습**: 전이 학습 또는 처음부터 학습

### 분석 결과

#### 출력 정보
- **생육 단계**: 현재 생육 단계 및 진행률
- **생육 지표**: 측정된 생육 지표 값
- **스트레스 지수**: 스트레스 정도 (0~100)
- **건강 점수**: 전체 건강 상태 점수 (0~100)

## 활용 사례

### 1. 생육 모니터링
- 일일 생육 상태 자동 기록
- 생육 이상 조기 발견
- 생육 예측 및 수확 시기 예측

### 2. 스트레스 대응
- 수분 스트레스 발견 → 자동 관수
- 영양 스트레스 발견 → 비료 공급
- 환경 스트레스 발견 → 환경 제어

### 3. 최적화
- 생육 조건 최적화
- 재배 방법 개선
- 수확량 증대

## 성능 지표

### 정확도
- **생육 단계 판별**: 90% 이상 정확도
- **스트레스 탐지**: 85% 이상 정확도
- **생육 지표 측정**: ±5% 오차 범위

### 처리 속도
- **이미지 분석**: 1~5초 내 분석 완료
- **실시간 처리**: 실시간 스트림 처리 가능
